# Open Source Scope

This repository is the public source tree for the PARA application workspace.

## In Scope

- the React Native app
- the web client surfaces in this workspace
- public docs for building, testing, and contributing
- patches and modules needed to run the public codebase

## Out of Scope

- secrets, tokens, signing certificates, and production credentials
- private deployment automation and internal-only sync jobs
- local caches, session databases, IDE state, and generated build output
- internal runbooks that expose operational details without helping public contributors

## Practical Rule

If a file helps an external developer understand, run, or improve the public app, it likely belongs here.

If a file exists mainly to operate private infrastructure or one machine, it likely does not.
